Project 2

FootyApp

This is a Web Application built in Java, Bootstrap, HTML, CSS, MySQL, and Postman that renders information from an API to provide users with live coverage on football(soccer) games from Europe’s most entertaining leagues such as the Champions League, LaLiga, Premier League, Bundesliga etc. During this project I performed extensive research on API’s and how to pull information from them, researched different APIs to find the better fitting for the project that was being built, created a simple and easy to read wireframe using Excalidraw and came up with detailed user stories using Trello as well as a detailed timeline to guide myself through this project to boost efficiency and productivity and developed a dashboard that is simple to use and provides all the needed information to keep up with a football(soccer) match.
